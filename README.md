# zcar2

![zcar2 image](image/zcar2.png)

ほぼ全パーツ3Dプリント製の4WD RCドリフトカー。1:28スケール、ホイールベース90mm（M）のMINI-Zシェルに対応\*。

\*注意: シャシーのフロント部分が比較的高いため、RX-7 FDなど一部のシェルには適合しません。AE86シェルでの動作を確認済みです。

テスト走行: https://youtu.be/90DoZpYbfy4

ユニバーサルジョイント クローズアップ: https://youtu.be/g_EzM2HMQvg

このデザインにはプリント難易度の高いパーツが含まれています。挑戦する前にプリンターのキャリブレーションが十分であることを確認してください。

## 特徴

- 4輪独立スライド式サスペンション
- プリント製ユニバーサルジョイントによる4WD駆動（デフなし）
- バッテリーとモーターをミッドマウント配置 — 前後重量配分は約50:50
- ドリフト向けパラレルステアリングジオメトリー
- 一般的なM3ネジ類を使用、すべてロックナット
- TPUプリントタイヤ

## ボディの選び方

zcar2のホイールベースは **90mm（Mサイズ）** です。京商 MINI-Z用ボディを購入する際は、必ず **ホイールベース90mm / M Narrow** と記載のあるものを選んでください。98mm（L）や94mm等のボディは大きすぎて合いません。

### 塗装済みボディ（オートスケールコレクション）

- [京商 MZQ101 頭文字D トヨタ スプリンター トレノ AE86](https://www.amazon.co.jp/dp/B07N354B1M) — 塗装済み・動作確認済み

### 未塗装ホワイトボディ（自分で塗装する場合）

- [京商 MZN214 トヨタ スプリンター トレノ AE86 ホワイトボディセット（AWD用ホイール付）](https://www.amazon.co.jp/dp/B0C52T1BF2) — ホイールベース90mm M Narrow
- [京商 MZN224 トヨタ レビン AE86 ホワイトボディセット（AWD用ホイール付）](https://www.amazon.co.jp/dp/B0D34J11WS) — ホイールベース90mm M Narrow

### 注意

- 京商 MZN198 Mercedes AMG GT3 等のホイールベース98mm（L）のボディは適合しません
- zcar2はシャシーのフロント部分が比較的高いため、RX-7 FDなど車高の低いシェルには適合しません
- ボディ購入時は必ずホイールベースの表記を確認してください

## Bambu Lab での印刷設定（A1シリーズ）

### 基準

- **ベースプリセット**: 0.12mm High Quality @BBL A1
- **基本素材**: PLA
- **考え方**:
  - **ギア**は精度と強度優先
  - **シャーシ**は軽さ優先
  - **シャフト・サス部品**は強度と軽さのバランス
- Bambu Studioでは、100%充填に使えるのは Rectilinear など一部のパターンで、Gyroid と Grid は100%非対応

### パーツ別設定

#### 1. ギア

| 項目 | 設定値 |
|------|--------|
| 積層ピッチ | 0.12mm |
| 壁面層数 | 5 |
| 充填密度 | 100% |
| 充填パターン | Rectilinear（直線） |

ギアは外周精度と歯の強度が重要なので、壁を厚くして100%で詰める。100%を使うなら Gyroid / Grid ではなく Rectilinear 系にする。

#### 2. モーターマウント

| 項目 | 設定値 |
|------|--------|
| 積層ピッチ | 0.12mm |
| 壁面層数 | 5 |
| 充填密度 | 80〜100% |
| 充填パターン | Gyroid（100%にするなら Rectilinear） |

最初は80% Gyroidで十分。

#### 3. シャフト・アクスル・ユニバーサルジョイント・細い機械部品

| 項目 | 設定値 |
|------|--------|
| 積層ピッチ | 0.12mm |
| 壁面層数 | 4 |
| 充填密度 | 80% |
| 充填パターン | Gyroid |

zcar2のユニバーサルジョイントは特に精度が求められるため、プリンターのキャリブレーションを十分に行うこと。

#### 4. サスペンション部品・ナックル・ハブ

| 項目 | 設定値 |
|------|--------|
| 積層ピッチ | 0.12mm |
| 壁面層数 | 4 |
| 充填密度 | 60% |
| 充填パターン | Gyroid |

#### 5. シャーシ本体

| 項目 | 設定値 |
|------|--------|
| 積層ピッチ | 0.12mm |
| 壁面層数 | 3 |
| 充填密度 | 30% |
| 充填パターン | Gyroid |

シャーシは全部を硬くするより、軽く作って必要な剛性だけ出す方が走りやすい。

### 充填パターンの使い分け

- **Gyroid**: 強度バランスが良く、ノズル干渉が起きにくい。RC部品向き
- **Grid**: シンプルで速いが、交差部分で擦れやすいことがある
- **Rectilinear**: 100%充填に対応。ギア専用

zcar / zcar2では**基本 Gyroid**でOK。ただし**ギアだけは100% Rectilinear**にする。

### 迷ったらこの3つだけ覚えればOK

- **ギア** → 0.12 / 壁5 / 100% / Rectilinear
- **機械部品（シャフト・マウント・サス・ユニバーサルジョイント）** → 0.12 / 壁4 / 60〜80% / Gyroid
- **シャーシ** → 0.12 / 壁3 / 30% / Gyroid

## zcar と zcar2 の比較

|                        | zcar                                           | zcar2                                              |
| ---------------------- | ---------------------------------------------- | -------------------------------------------------- |
| **駆動方式**           | リア駆動（ギアトレイン）                       | 4WD（プリント製ユニバーサルジョイント）             |
| **ステアリング**       | アッカーマン（最大45°）                        | パラレル（ドリフト向け）                           |
| **サスペンション**     | 独立フロント＋ヒンジ式リア                     | 4輪独立スライド式                                  |
| **重量配分**           | リア寄り（モーター後方配置）                   | 前後約50:50（ミッドマウント）                      |
| **ベアリング**         | 625（5x16x5mm）× 4個                          | MR105（5x10x4mm）× 8個                            |
| **サーボ**             | SG90（入手しやすい汎用品）                     | GH-S37D 3.7g（小型・軽量）                         |
| **タイヤ**             | MINI-Z用タイヤ（別途購入）                     | TPUプリント（3Dプリンターで製作）                  |
| **ギア比**             | 4種類から選択可能                              | 固定                                               |
| **ネジの種類**         | 3種類の長さのみ                                | 4種類の長さ                                        |
| **プリント難易度**     | 普通                                           | 高い（キャリブレーション必須）                     |
| **組み立てやすさ**     | 比較的簡単（パーツ点数が少ない）               | やや難しい（ユニバーサルジョイント等の精密パーツ） |
| **シェル適合性**       | 幅広いMINI-Zシェルに対応                       | フロントが高く一部シェル非対応（RX-7 FD等）        |
| **向いている用途**     | グリップ走行・初めてのビルドに                 | ドリフト走行・4WDの走破性を求める場合に            |

### zcarのメリット
- パーツ点数が少なく組み立てが簡単
- プリント難易度が低く失敗しにくい
- SG90サーボや625ベアリングなど入手しやすいパーツを使用
- ギア比を4種類から選べるので走行スタイルに合わせやすい

### zcarのデメリット
- リア駆動のため4WDに比べてトラクションが劣る
- タイヤを別途購入する必要がある

### zcar2のメリット
- 4WD駆動で高いトラクションと走破性
- 前後50:50の重量配分でドリフトに最適
- タイヤもTPUで3Dプリントできる（追加購入不要）
- 4輪独立サスペンションで走行性能が高い

### zcar2のデメリット
- プリント難易度が高く、プリンターの精度が求められる
- ユニバーサルジョイント等の精密パーツの組み立てが難しい
- 一部のMINI-Zシェルが適合しない
- GH-S37D サーボはSG90ほど一般的でない
